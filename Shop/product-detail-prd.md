# 农资商城产品详情页 - 产品需求文档

## 1. 产品概述

### 1.1 文档目的
本文档定义了农资商城产品详情页的功能需求、非功能需求、用户体验设计规范和技术实现标准，作为产品开发、测试和验收的依据。

### 1.2 产品背景
农资商城是一家专注于农业生产资料的电商平台，为农民、农业合作社和经销商提供优质的农药、化肥、种子等农业生产资料。产品详情页是连接用户与商品的关键界面，直接影响用户购买决策和转化率。

### 1.3 目标用户
- **终端用户**：农民、种植户，关注产品效果、价格和使用方法
- **经销商/代理商**：关注代理政策、区域保护和利润空间
- **农业合作社**：关注批量采购价格和售后服务

### 1.4 核心价值
- 提供全面、清晰的产品信息展示
- 优化用户购买决策流程
- 支持多规格商品选择和灵活购买
- 实现区域保护机制，保障代理商权益
- 提升用户信任度和购买转化率

## 2. 用户故事

### 2.1 终端用户故事

#### 2.1.1 浏览产品详情
**作为**一名种植户，
**我希望**能够查看详细的产品信息（包括图片、描述、规格、使用方法等），
**以便**了解产品是否符合我的需求。

**验收标准**：
- 能够查看至少5张产品图片，包括主图、细节图和使用场景图
- 能够切换查看不同规格的产品信息
- 能够看到产品的详细描述、特点和使用方法
- 能够查看其他用户的评价和评分

#### 2.1.2 选择产品规格并购买
**作为**一名种植户，
**我希望**能够选择合适的产品规格和数量，
**以便**购买到满足我种植需求的产品。

**验收标准**：
- 能够选择不同的产品规格（如50kg/包、25kg/包、10kg/包）
- 能够调整购买数量（至少1件，无上限）
- 能够将产品加入购物车或立即购买
- 购买操作流畅，无明显延迟

#### 2.1.3 了解产品资质
**作为**一名种植户，
**我希望**能够查看产品的相关资质证明，
**以便**确认产品的合法性和质量保障。

**验收标准**：
- 能够查看农药登记证、生产许可证、产品标准证等资质文件
- 资质文件清晰可辨，包含有效期等关键信息

### 2.2 经销商/代理商故事

#### 2.2.1 查看代理政策
**作为**一名潜在经销商，
**我希望**能够查看产品的代理政策和区域保护信息，
**以便**评估是否适合成为该产品的代理商。

**验收标准**：
- 能够看到产品的代理区域限制信息
- 能够查看代理政策的基本要求和权益
- 能够查看当前区域是否已有代理商

#### 2.2.2 申请代理/经销资格
**作为**一名潜在经销商，
**我希望**能够在线申请产品的代理或经销资格，
**以便**成为该产品的合法销售渠道。

**验收标准**：
- 能够点击"申请代理"或"申请经销商"按钮
- 能够看到区域保护提示（如果当前区域已有代理）
- 能够获取当地现有代理商的联系方式

## 3. 功能需求

### 3.1 产品信息展示

#### 3.1.1 图片展示
- **功能**：展示产品的多角度图片
- **规格**：
  - 主图尺寸：移动端300px-400px高度，桌面端500px-600px高度
  - 缩略图：移动端水平滚动，桌面端垂直排列
  - 支持至少5张产品图片
- **交互**：点击缩略图切换主图，支持图片放大效果

#### 3.1.2 基本信息
- **功能**：展示产品的核心信息
- **内容**：
  - 产品标题
  - 评分和评价数量
  - 销量信息
  - 价格（原价、优惠价）
  - 限时优惠标识

#### 3.1.3 店铺信息
- **功能**：展示销售该产品的店铺信息
- **内容**：
  - 店铺名称和logo
  - 店铺评分（综合评分、物流评分、服务评分）
  - 店铺标签（如"厂家直销"、"官方认证"）
  - 进入店铺按钮

### 3.2 购买功能

#### 3.2.1 规格选择
- **功能**：支持选择不同规格的产品
- **规格**：
  - 支持3种以上规格选择
  - 选中状态清晰可见
  - 显示当前规格的库存状态

#### 3.2.2 数量控制
- **功能**：调整购买数量
- **规格**：
  - 支持+/-按钮调整数量
  - 支持直接输入数字
  - 最小数量为1

#### 3.2.3 购买操作
- **功能**：提供多种购买方式
- **按钮**：
  - 加入购物车
  - 立即购买
  - 申请代理
  - 申请经销商

### 3.3 区域保护机制

#### 3.3.1 区域限制检查
- **功能**：根据用户位置检查是否允许代理申请
- **逻辑**：
  - 获取用户当前位置（省/市）
  - 与产品代理区域进行匹配
  - 如果用户所在区域已有代理，显示限制提示

#### 3.3.2 本地推荐
- **功能**：为受限用户推荐本地代理商
- **内容**：
  - 显示本地代理商名称、联系方式
  - 提供联系代理商的便捷方式

### 3.4 标签页内容

#### 3.4.1 商品详情
- **功能**：展示产品的详细描述
- **内容**：
  - 产品概述
  - 产品特点（列表形式）
  - 使用场景图片
  - 适用作物和使用方法

#### 3.4.2 买家评价
- **功能**：展示用户对产品的评价
- **内容**：
  - 评价总数
  - 用户头像、昵称、评价时间
  - 评分星级
  - 评价内容

#### 3.4.3 商品属性
- **功能**：展示产品的详细属性
- **内容**：
  - 品牌、类别、规格
  - 产地、毒性、剂型
  - 有效成分、含量
  - 使用作物、防治对象
  - 生产许可证、农药登记证等
- **格式**：采用网格布局，每个属性独立卡片展示

#### 3.4.4 商品资质
- **功能**：展示产品的相关资质证明
- **内容**：
  - 农药登记证
  - 生产许可证
  - 产品标准证
  - 质量认证（如ISO9001）
- **格式**：每个资质包含证书信息和证书图片

#### 3.4.5 商品描述
- **功能**：提供更详细的产品介绍
- **内容**：
  - 产品研发背景
  - 产品优势
  - 使用建议
  - 更多使用场景图片

### 3.5 辅助功能

#### 3.5.1 联系客服
- **功能**：提供与客服沟通的入口

#### 3.5.2 分享功能
- **功能**：支持将产品分享给他人

#### 3.5.3 相关推荐
- **功能**：展示与当前产品相关的其他商品
- **规格**：
  - 移动端2列布局，桌面端4列布局
  - 每个推荐商品包含图片、标题、价格、销量

## 4. 非功能需求

### 4.1 性能要求
- 页面加载时间：移动端≤3秒，桌面端≤2秒
- 图片加载优化：使用CDN加速，支持懒加载
- 交互响应时间：所有按钮点击和页面切换响应时间≤500ms

### 4.2 兼容性要求
- 浏览器兼容：Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- 移动设备兼容：iOS 13+、Android 8+
- 屏幕适配：支持320px-1920px宽度的屏幕

### 4.3 可用性要求
- 符合WCAG 2.1 AA级无障碍标准
- 操作流程简单直观，减少用户学习成本
- 错误提示清晰明确，指导用户正确操作

### 4.4 安全性要求
- 用户位置信息加密存储和传输
- 防止SQL注入和XSS攻击
- 敏感信息（如价格、库存）进行服务器端验证

### 4.5 可维护性要求
- 代码结构清晰，使用模块化设计
- 充分的代码注释和文档
- 支持功能扩展和版本迭代

## 5. 用户体验设计

### 5.1 视觉设计
- **主色调**：使用绿色系（#10b981）作为主色调，体现农业和环保理念
- **辅助色**：使用橙色系（#f59e0b）作为行动按钮颜色，增强视觉吸引力
- **排版**：采用清晰的层级结构，标题使用较大字号和粗体，正文保持良好的可读性
- **图标**：使用Font Awesome图标库，保持图标风格统一

### 5.2 交互设计
- **图片交互**：点击缩略图切换主图，支持平滑过渡动画
- **按钮交互**：所有可点击元素有明确的悬停效果，按钮点击有反馈
- **模态框**：区域保护提示使用模态框，支持平滑的显示和隐藏动画
- **标签页**：标签切换时内容区域有淡入效果，当前标签有明显标识

### 5.3 响应式设计
- **移动端**：垂直布局，简化导航，突出核心购买功能
- **平板**：平衡展示和操作，优化图片尺寸
- **桌面端**：充分利用屏幕空间，提供更丰富的信息展示

## 6. 技术实现

### 6.1 前端技术栈
- HTML5 + CSS3 + JavaScript
- Tailwind CSS 3.x（样式框架）
- Font Awesome 6.x（图标库）

### 6.2 核心功能实现

#### 6.2.1 图片切换
```javascript
// 缩略图切换主图
const thumbnails = document.querySelectorAll('.thumbnail');
const mainImage = document.querySelector('.product-image');

thumbnails.forEach(thumbnail => {
    thumbnail.addEventListener('click', function() {
        thumbnails.forEach(t => t.classList.remove('active'));
        this.classList.add('active');
        const img = this.querySelector('img');
        mainImage.src = img.src.replace('150x150', '800x600');
    });
});
```

#### 6.2.2 标签页切换
```javascript
// 标签页切换
const tabBtns = document.querySelectorAll('.tab-btn');
const tabContents = document.querySelectorAll('.tab-content');

tabBtns.forEach(btn => {
    btn.addEventListener('click', function() {
        const tabId = this.getAttribute('data-tab');
        
        // 更新标签按钮状态
        tabBtns.forEach(b => {
            b.classList.remove('text-emerald-600', 'border-b-2', 'border-emerald-600');
            b.classList.add('text-gray-600');
        });
        this.classList.remove('text-gray-600');
        this.classList.add('text-emerald-600', 'border-b-2', 'border-emerald-600');
        
        // 切换标签内容
        tabContents.forEach(content => {
            content.classList.add('hidden');
        });
        document.getElementById(`tab-${tabId}`).classList.remove('hidden');
    });
});
```

### 6.3 图片资源
- 使用Picsum.photos作为图片CDN，确保图片加载稳定
- 为每个产品图片设置固定seed参数，保证图片一致性
- 图片尺寸：缩略图150x150px，主图800x600px

## 9. 风险与依赖

### 9.1 风险
- **图片加载失败**：依赖第三方CDN，存在加载失败风险
- **区域定位不准确**：用户位置获取可能存在偏差
- **移动端性能**：图片和动画可能影响移动端性能

### 9.2 依赖
- **第三方服务**：Picsum.photos图片CDN
- **地理位置服务**：浏览器定位API
- **后端接口**：产品信息、库存、价格等数据接口

## 11. 附录

### 11.1 术语定义
- **农资**：农业生产资料，包括农药、化肥、种子等
- **区域保护**：为保护代理商权益，限制同一区域内的代理数量
- **经销商**：从厂家或代理商处进货，再销售给终端用户的商家
